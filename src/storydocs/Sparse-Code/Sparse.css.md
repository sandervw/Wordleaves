# The Laws of Sparse CSS

## The Primary Laws

1. All Sparse CSS Laws must adhere to the Laws defined in `Sparse.md`. `No Exceptions.`
2. All CSS specific Sparse Laws must be specified within this file. `No Exceptions.`

## CSS Specific Laws

3. There are no more than 5 of any variable type. `No Exceptions.`
4. There are no hardcoded numeric values - only CSS variables. `No Exceptions.`
5. There are no ems - only rems. `No Exceptions.`
6. There are no more than 3 distinct 'types' of a class. `No Exceptions.`
7. There are 4 percentages: 25%, 50%, 75%, 100%. `No Exceptions.`
8. There are 2 colors: 'copper' and 'danger'. `No Exceptions.`
9. There may be no more than 2 classes applied to any element. `No Exceptions.`
10. There is only one animation speed. `No Exceptions.`
11. There is only one border radius. `No Exceptions.`
12. No box shadows. `No Exceptions.`

`No Exceptions.`
